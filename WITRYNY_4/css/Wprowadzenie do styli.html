Wprowadzenie do styli - ćwiczenie 1

Kaskadowe Arkusze Stylów to język służący do opisu formy prezentacji stron WWW

Style można dołączać na trzy sposoby
  • jako osobny plik - tzw. zewnęrzny arkusz CSS
  • osadzając je między tagami <style> i </style> koniecznie umieszczonymi wewnątrz sekcji <head>
  • umieszczając je wewnątrz atrybutu znacnzika - tzw style inline

Zewnętrzny arkusz stylów zawiera CSS w oddzielnym pliku z .css rozszerzeniem. Jest to najczęstsza i naprzydatniejsza metoda wprowadzania CSS do dokumentu. Możesz polączyć pojedynczy plik CSS z wieloma stronami internetowymi, stylizując je wszystkie samym arkuszem stylów CSS.
Odwołanie do zewnętrznego arksuza stylów CSS odbywa się za pomocą tagu < link > z atrybutem href < link rel="stylesheet" href="nazwa arkusza wraz ze sciażką dostępu" >

Wewnętrzny arkusz stylów znajduje się w dokumencie HTML.
Aby utworzyć wewnętrzny arkusz stylów, należy umieścić CSS wewnątrz <style>elementu zawartego w HTML <head>

Style inline to deklaracje CSS, które wpływają na pojedynczy element HTML, zawarty w atrybucie style.
< p style="color:red;">This is my first CSS example </p>
